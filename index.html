HTML5

HTML5 is the fifth version of the HTML programming language (HyperText Markup Language) that improves and takes advantage of the full potential of the Word Wide Web (WWW). To understand in a simple way what HTML5 consists of, we must explain that it offers functionalities that improve the browsing experience, that is, the different browsers (Chrome, Explorer, Safari or Firefox) quickly understand how a website is structured and what are the elements that make it up. make up, although this was already incorporated in the previous version, new elements can now be included, reducing the use of plugins and improving loading speed.

CSS

CSS is a style sheet language created to control the appearance or presentation of electronic documents defined with HTML and XHTML. CSS is the best way to separate content and presentation and is essential for creating complex web pages.Separating the definition of the content and the definition of its appearance has numerous advantages, since it forces to create well-defined HTML/XHTML documents with complete meaning (also called "semantic documents"). In addition, it improves the accessibility of the document, reduces the complexity of its maintenance and allows the same document to be viewed on an infinite number of different devices.

JAVASCRIPT

JavaScript is a programming language that developers use to make web pages interactive. From updating social media feeds to displaying interactive maps and animations, JavaScript features can enhance a website's user experience. As a server-side scripting language, it is one of the main technologies of the World Wide Web. For example, when browsing the Internet, anytime you see an image carousel, a “click-to-show” drop-down menu, or dynamically changing color elements on a web page, you will be seeing the effects of JavaScript.
